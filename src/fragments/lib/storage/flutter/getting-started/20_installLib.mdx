Add the following dependency to your **app**'s `pubspec.yaml` along with others you added above in **Prerequisites**:

```yaml
environment:
  sdk: ">=2.17.3 <3.0.0"

dependencies:
  flutter:
    sdk: flutter
  amplify_flutter: ^0.5.1
  amplify_auth_cognito: ^0.5.1
  amplify_storage_s3: ^0.5.1
```

Don't forget to import the libraries in the necessary places of your project:

```dart
import 'package:amplify_flutter/amplify_flutter.dart';
import 'package:amplify_auth_cognito/amplify_auth_cognito.dart';
import 'package:amplify_storage_s3/amplify_storage_s3.dart';
```
