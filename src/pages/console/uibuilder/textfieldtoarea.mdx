  description: `How to convert existing TextField primitive components to TextAreaField primitive components`

After [upgrading](/console/uibuilder/figmatocode/#upgrading-figma-files) to the current Figma community file. You can update any existing TextFields into TextAreaFields by to the instance of the containing component and changing the TextField instance to the TextAreaField component on the primitives page.
